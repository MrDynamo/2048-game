## Contributing

Requirements:

    - Python v3.12.0
    - node v21.7.1
    - npm 10.5.0

### Frontend Requirements

```cd frontend && ./setup-dev-frontend.sh```

OR

```cd frontend && npm install```

### Backend Requirements

```cd backend && ./setup-dev-backend.sh```

OR

```cd backend && source 2048-game/bin/activate && pip install -r requirements.txt```
